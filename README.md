# 📖 About the Project

ShopEZ is an AI-powered online shopping platform developed to make shopping easier, faster, and more personalized. Unlike traditional e-commerce websites, ShopEZ includes an intelligent AI assistant that helps users discover products, compare options, and receive instant shopping guidance.

The application focuses on providing a clean, responsive, and user-friendly interface while leveraging AI to improve the overall shopping experience.

Whether you're looking for the perfect product or need buying suggestions, ShopEZ helps you make smarter shopping decisions.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/Sathwika0412/Shop-EZ.git
```

---

## Navigate to the Project

```bash
cd Shop-EZ
```

---

## Install Dependencies

```bash
npm install
```

---

## Configure Environment Variables

Create a `.env` file using `.env.example`.

Example:

```env
VITE_FIREBASE_API_KEY=YOUR_API_KEY
VITE_FIREBASE_AUTH_DOMAIN=YOUR_AUTH_DOMAIN
VITE_FIREBASE_PROJECT_ID=YOUR_PROJECT_ID
VITE_FIREBASE_STORAGE_BUCKET=YOUR_STORAGE_BUCKET
VITE_FIREBASE_MESSAGING_SENDER_ID=YOUR_SENDER_ID
VITE_FIREBASE_APP_ID=YOUR_APP_ID
```

---

## Start the Development Server

```bash
npm run dev
```
